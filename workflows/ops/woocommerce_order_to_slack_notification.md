# New WooCommerce order to Slack

## What it does
Sends a Slack notification when a new WooCommerce order is created.

## Trigger
- Order Created (wooCommerceTrigger)

## Systems / Nodes
- Slack
- WooCommerce

## Setup
- Configure WooCommerce Trigger credentials.
- Configure Slack credentials/channel and message format.

## Notes
- Keep this as an ops add-on; your main portfolio value should be the ERP/finance sync workflows.
